# Senior Forge Codex
> The definitive technical reference for the path to Android Seniority.

The **Senior Forge Codex** is a strategic compendium designed to document the deep theory required for Android mastery. It serves as the permanent theoretical foundation for the concepts explored in **The Senior Forge** articles and as a comprehensive study guide for **Senior Android Developer technical interviews**.

## 🚀 Purpose
Documentation is the cornerstone of seniority. This Codex transforms abstract engineering principles into a structured reference manual, covering everything from Kotlin internals to Artificial Intelligence in mobile ecosystems.

## 🛠 Tech Stack
- **Engine:** [Jekyll](https://jekyllrb.com/)
- **Theme:** [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) (Modernized & Customized)
- **Deployment:** GitHub Actions & GitHub Pages
- **Styling:** Sass (Dart Sass compliant)

## 🌍 Bilingual Infrastructure
The Codex is built with a fully localized, context-aware structure:
- **English (`/`)**: Main technical reference.
- **Español (`/es/`)**: Edición completa en español.
- **Context-Aware Switching**: Dynamic language toggle flags that maintain your current reading position.

## 📚 Codex Structure
The knowledge is organized into 10 strategic chapters:
1. **Kotlin Core**
2. **Coroutines & Flow**
3. **Jetpack Compose**
4. **Android Framework & Lifecycle**
5. **Architecture & Design Patterns**
6. **Dependency Injection**
7. **Networking & Data**
8. **Testing & CI/CD**
9. **Performance & Security**
10. **Artificial Intelligence**

## 💻 Local Development

Ensure you have Ruby and Bundler installed, then run:

```bash
# Install dependencies
bundle install

# Start the local server
bundle exec jekyll serve
```

Access the site at `http://localhost:4000`.
