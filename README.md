# Senior Forge Codex

> The definitive technical reference for the path to Android Seniority.

**Live Codex:** [https://aghmnl.github.io/senior-forge-codex/](https://aghmnl.github.io/senior-forge-codex/)

The **Senior Forge Codex** is a strategic compendium designed to document the deep theory required for Android mastery. It serves as the permanent theoretical foundation for the concepts explored in **The Senior Forge** articles and as a comprehensive study guide for **Senior Android Developer technical interviews**.

## 🚀 Purpose

Documentation is the cornerstone of seniority. This Codex transforms abstract engineering principles into a structured reference manual, covering everything from Kotlin internals to Artificial Intelligence in mobile ecosystems. It reflects a journey of professional growth and the commitment to high-level software craftsmanship.

## 🛠 Tech Stack

- **Engine:** [Jekyll](https://jekyllrb.com/)
- **Theme:** [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) (Modernized & Customized)
- **Deployment:** GitHub Actions & GitHub Pages
- **Styling:** Sass (Dart Sass compliant)

## 🌍 Bilingual Infrastructure

The Codex is built with a fully localized, context-aware structure:

- **English (`/`)**: Main technical reference and global standards.
- **Español (`/es/`)**: Edición completa en español con terminología técnica adaptada.

## 📚 Codex Structure

The knowledge is organized into 10 strategic chapters, each covering essential domains for senior-level proficiency:

1. **Kotlin Core**: Advanced language features and functional programming.
2. **Coroutines & Flow**: Mastery of asynchronous streams and structured concurrency.
3. **Jetpack Compose**: Modern UI development, stability, and state management.
4. **Android Framework & Lifecycle**: Deep understanding of the OS components and process death.
5. **Architecture & Design Patterns**: MVI, Clean Architecture, and SOLID principles.
6. **Dependency Injection**: Scalable DI using Hilt and Dagger 2 concepts.
7. **Networking & Data**: Robust data persistence and API communication strategies.
8. **Testing & CI/CD**: High-quality assurance and automated deployment pipelines.
9. **Performance & Security**: Profiling, memory management, and data encryption.
10. **Artificial Intelligence**: LLM integration, Agents, and on-device AI patterns.

## 💻 Local Development

To run the Codex locally for previewing changes, ensure you have Ruby and Bundler installed, then execute:

```bash
# Install dependencies
bundle install

# Start the local server
bundle exec jekyll serve
```
